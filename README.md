## Featured Projects

### Enterprise RAG System

Production-style Retrieval-Augmented Generation backend for e-commerce support.
It uses Supabase pgvector for semantic retrieval, LangChain orchestration,
multi-provider LLM routing, prompt-safety checks, and a Gradio testing UI.

Repository: [RAG-System](https://github.com/malheasa0t-prog/RAG-System)

### Tensar

Full-stack e-commerce and services platform built with React, Vite, Supabase,
Cloudflare Functions, GitHub Actions, and a Groq-powered AI chat agent.

Repository: [tensar](https://github.com/malheasa0t-prog/tensar)

### Phone Damage Detection

YOLO-based computer vision project for detecting phone condition and damage
classes from images, with reusable augmentation, duplicate cleanup, training,
and inference scripts.

Repository: [PHONE-](https://github.com/malheasa0t-prog/PHONE-)

### Car and Food CLIP Model

Fine-tuned CLIP image-text retrieval project with caption preparation, training,
and Gradio demo utilities for image-to-text and text-to-image search.

Repository: [car-and-food-clip-model](https://github.com/malheasa0t-prog/car-and-food-clip-model)
